# Node typescript boilerplate

# Usage

## Installing
To install the application, run the following command:
```
npm install
```

## Building
### Development

```
npm run build:dev
```

### Production

```
npm run build
```

## Running
### Development

```
npm run run:dev --logfile myapp-dev.log
```

### Production

```
npm run --logfile myapp-prod.log
```
